# Hifadih BI Chart Plugins

Every chart in this directory is a standalone module (workspace).
They are generated as independent entities to allow for modular scaling and easy migration.

## How to add a new chart
To simulate the generation of a new chart with npx:
`npx hifadih-cli add chart [name]`

## Available Plugins
- bar
- line
- pie
- radar
- scatter
- heatmap
- sankey
- treemap
