# 🚀 BI-Plateforme

**A Next-Generation, Human-Centered, AI-Ready Business Intelligence System.**

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen.svg)](https://saintlaurent00.github.io/BI-Plateforme/)

## 🧠 The Vision

BI-Plateforme is NOT just another dashboard tool. It is a **low-complexity BI system** designed to outperform traditional tools by focusing on radical simplicity, guided UX flows, and motion-driven insights.

Traditional BI (like Apache Superset) is often overwhelming for non-technical users. BI-Plateforme reframes the problem: **Users don’t want “tools”, they want answers in seconds.**

## ✨ Key Differentiators

*   **10x Simpler UX:** Guided 3-step chart building (Dataset → Metric → Dimension).
*   **Query Intelligence Engine:** Logical BI requests are dynamically translated to optimized SQL.
*   **Motion-Driven Experience:** Powered by `anime.js` v4 to guide attention and reduce cognitive load.
*   **AI-Ready:** Built-in semantic mapping and heuristic insight generation.
*   **Real-time Visual Feedback:** Instant preview system for every configuration change.
*   **Zero SQL Exposure:** Powerful analytics without ever writing a line of code.

## 🏗️ Architecture

### Backend: Query Intelligence Engine
*   **Dynamic Query Generation:** Advanced `QueryBuilder` class for SELECT, FROM, WHERE, and GROUP BY logic.
*   **Dataset Abstraction Layer:** Semantic mapping of tables into logical BI entities.
*   **Insight Engine:** Heuristic-based analysis to highlight trends and anomalies automatically.

### Frontend: Next.js + anime.js
*   **Guided Chart Builder:** A wizard-based flow that ensures success for non-technical users.
*   **Simple Dashboard Builder:** Intuitive grid-based layout for assembling insights.
*   **Motion Design System:** Meaningful transitions that explain state changes and data reveals.

## 🛠️ Tech Stack

*   **Framework:** React (Vite)
*   **Styling:** Tailwind CSS
*   **Motion:** anime.js v4
*   **Database:** sql.js (WASM SQLite) for client-side query intelligence
*   **Icons:** Lucide React
*   **Notifications:** Sonner

## 🚀 Getting Started

1.  **Clone the repository**
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run locally:**
    ```bash
    npm run dev
    ```
4.  **Build for production:**
    ```bash
    npm run build
    ```
